Gyerünk, tanítsuk meg a robotjainkat megkülönböztetni a szavakat a számoktól!

Adott egy karakterlánc, amely szóközökkel elválasztott szavakat és számokat tartalmaz (minden szám és/vagy szó között csak egy szóköz van). A szavak csak betűkből állnak. Önnek le kell ellenőriznie, hogy van-e az eredeti szövegben három szó egymás után. Például, a "start 5 one two three 7 end" karakterlánc tartalmaz három egymás utáni szót.

Bemeneti adatok: egy szavakat tartalmazó karakterlánc (str).

Kimeneti adatok: a felelet logikai kifejezésként (bool), True vagy False.

Példák: 

{% if interpreter.slug == "js-node" %}
threeWords("Hello World hello") == True
threeWords("He is 123 man") == False
threeWords("1 2 3 4") == False
threeWords("bla bla bla bla") == True
threeWords("Hi") == False
{% else %}
checkio("Hello World hello") == True
checkio("He is 123 man") == False
checkio("1 2 3 4") == False
checkio("bla bla bla bla") == True
checkio("Hi") == False
{% endif %}

Alkalmazása: Ez a feladat segít a szöveges típusú adatok feldolgozásában és bemutat néhány hasznos függvényt is.

Előfeltételek:
Az eredeti szöveg csak szavakat és/vagy számokat tartalmaz. Vegyes tartalmú szavak nincsenek (például betűk és számjegyek keverve egy szóban).
