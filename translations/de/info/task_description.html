Lass uns den Robotern beibringen Worte und Zahlen zu unterscheiden.

Gegeben wird ein String mit Worten und Zahlen, die durch ein Leerzeichen getrennt sind. Die Worte enthalten nur Buchstaben. Du sollst prüfen, ob der String drei Worte nacheinander enthält. Zum Beispiel: "Start 5 eins zwei drei 7 Ende", dieser String enthält drei Wörter in Folge.

Eingabe: Ein String mit Worten und Zahlen.

Ausgabe: Die Antwort als boolescher Wert. (True oder False)

Beispiele: 

checkio("Hello World hello") == True
checkio("He is 123 man") == False
checkio("1 2 3 4") == False
checkio("bla bla bla bla") == True
checkio("Hi") == False

Wozu das gut ist: In dieser Aufgabe lernst Du den Umgang mit Strings, außerdem werden einig nützliche Funktionen vorgestellt.

Bedingungen: Der vorgegebene String enthält Worte und/oder Zahlen. Ein Mix aus Buchstaben und Ziffern kommt nicht vor.
