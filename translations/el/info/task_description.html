Ας μάθουμε στα ρομπότ να ξεχωρίζουν λέξεις από αριθμούς.

Σου δίνεται μια πρόταση που περιέχει λέξεις και αριθμούς χωρισμένα με κενό (ένα διάστημα). Οι λέξεις περιέχουν μόνο γράμματα. Σκοπός είναι να εξετάσεις αν η πρόταση περιέχει τρεις λέξεις στη σειρά. Για παράδειγμα, η πρόταση "start 5 one two three 7 end" περιέχει τρεις λέξεις στη σειρά.

Είσοδος: Μια πρόταση σε μορφή string.

Έξοδος: Η απάντηση με τη μορφή λογικής τιμής.

Παράδειγμα:

threeWords("Hi") == False

Πώς χρησιμοποιείται: Η αποστολή αυτή θα σου μάθει πώς να δουλεύεις με strings και αποτελεί μια εισαγωγή σε χρήσιμες συναρτήσεις.

Προϋποθέσεις: Η είσοδος περιέχει λέξεις και/ή αριθμούς. Δεν υπάρχουν λέξεις που να περιέχουν και γράμματα και ψηφία.
